This project began as a 3D rocket CAD model designed for the Hack Club Stardance challenge, but navigating the platform's active tracking requirements and browser environment constraints triggered a complete structural pivot into software engineering. I initially attempted to build a server-side stock report maker using Python's yfinance library, but running it natively inside a client-side browser caused immediate CORS policy blockades, while switching to standard JavaScript API structures exposed private keys and hit strict rate limits. To keep the code completely open-source, keyless, and infinitely scalable for deployment on GitHub Pages, I re-engineered the architecture to pipe institutional-grade historical data directly from the exchange using the TradingView embedded terminal matrix. To deliberately destroy the generic aesthetic common to standard AI layouts, I built a rugged, high-density cybernetic terminal interface utilizing pure CSS scanline overlays, local storage session portfolio ledgers, a custom algorithmic volatility radar, and raw web browser AudioContext synthesis to generate reactive hardware sound waves without any external assets.
